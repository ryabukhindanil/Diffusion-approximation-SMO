Программный продукт и код приложения по дипломной работе "Диффузионная аппроксимация нормированных очередей систем массового обслуживания"

SMO_Diffusion - исполняемый exe файл для запуска и работы с моделями СМО.
diff_app - код приложения, предоставляемый на языке программирования высокого уровня Python.

Доступ к приложению можно найти в Release.
